# SecureTrack - Cyber Security Incident Reporting System

## 📌 Project Description
SecureTrack is a web-based Cyber Security Incident Reporting and Tracking System that allows users to report cyber incidents such as phishing, malware, and unauthorized access, and track their status through a secure dashboard.

## ⚙️ Technologies Used
- HTML
- CSS
- JavaScript
- (Future: Node.js, Express.js, MongoDB - MERN Stack)

## 🎯 Features
- User Registration and Login UI
- Incident Reporting Form
- Dashboard Interface
- Incident Type Classification
- Simple Client-side Validation

## 🚀 Future Improvements
- Backend API with Node.js & Express
- Database integration (MongoDB)
- JWT Authentication
- Admin Dashboard
- Email Notifications
